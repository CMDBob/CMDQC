README

The QCSource folder contains source code for progs.dat.

The ModelSource folder contains  all the model source stuff, like Max 9 files and textures.

The MapSource folder contains the Worldcraft 1.6 RMFs and the exported MAPs ready for compiling.

The TexSource folder contains raw PNG images waiting for Wally to insert into Quake WADs for Worldcraft.

The TexWads folder contains the compiled texture WADs.

The CMDMod folder has the 'finished' files ready for Quake.

SO yeah.
